# Withdrawing Funds

There are two different withdrawal ideas in Stream Engine:

## Payment Streams

For API payment streams, the recipient withdraws claimable value from the stream contract according to the stream rules.

## RWA Yield Streams

For productive RWAs, yield is claimed through the asset-linked stream flow. Future unclaimed yield follows current NFT ownership.

## Important

Claiming yield is not the same thing as transferring the NFT. Ownership controls future entitlement, not already claimed value.
